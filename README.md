# Simple-Speedcube-Timer-Application
Another project,  a Speedcube Timer desktop application built with PySide6 and SQLite3.

**Features**
-- Scramble generator # (Not the WCA official scramble)
-- 4 Puzzles Available # (2x2, 3x3, 4x4, 5x5 only)
-- Customizable Time Limit # (You can toggle time limit if you want and a sound will play if you solve it under and over the time limit)
-- Tracks Personal Records # (Singles best, Ao5 best, Ao12 best, Ao100 best)
-- Penalty Management # (You can penalized your solve if it has plus 2 penalty or dnf)
-- Strict Inspection Time # (You can not start the timer if you go over the inspection time limit (DNF displayed). It will be recorded as DNF)

**Built with**
-- Python
-- PySide6
--Pygame
--Sqlite3

**Modules to be installed**
-- PySide6
-- pygame
